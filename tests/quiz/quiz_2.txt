1. Как включить автосохранение данных в VSCODE?
Наверху окна рядом со значком VS Code есть панель, нажимаем File -> Auto Save

2. Как настроить перенос строки? 
File -> Preferences -> Settings. Далее в строке Search Settings пишем wrap. В первом выпадающем окне меняем off на wordWrapColumn, во втором пишем 79 (обозначаем максимальную длину строки в 79 символов).

4. Какие способы переноса строк показаны в видео:

4.4 Использование круглых скобок для продолжения строки

    "This is a very long string that I would like to "
    "continue on the next line."

4.6 Сложение строк с помощью +

6. Где в vscode находится клиент гита? как в нём отправить коммит? как принять домашку?
Клиент гита находится в боковой панель слева. Написать сообщение для коммита в поле Message, нажать на стрелочку на кнопке Commit и выбрать Commit&Push или отдельно сначала Commit, а потом Push.
Принять домашку - More Actions -> Pull.

GIT - это система управления версиями.
Да, он локальный. При этом код при коммите копируется в облако.
Да, в нём можно посмотреть историю изменений файлов, отменить изменения и откатиться назад на нужный коммит.

24. Что значит отвалился интерпритатор?
Если интерпритатор отвалился, это значит, что код не может быть выполнен. Такое может возникнуть если, например, в коде есть ошибки, недостаток памяти или других системных ресурсов для выполнения программы, конфликты зависимостей, неправильно установлен интерпретатор.
